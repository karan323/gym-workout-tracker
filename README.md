# Gym Workout Tracker

An endless day-by-day gym workout tracker with checklists for every muscle group.

Open `index.html` in your browser to use it.

## Features
- Chest, Back, Shoulders, Legs, Biceps, Triceps, Abs, Cardio
- Sets, reps and weight notes per exercise
- Day streak and workout stats
- Auto workout split by day of week
